# Student Data Management Using Linked List

## Description
This project is a menu-driven Student Data Management System implemented in C using a Singly Linked List. It allows users to manage student records efficiently through dynamic memory allocation.

## Features
- Add Student Record
- Display Student Records
- Delete a Student Record
- Modify Student Details
- Sort Records by Name or Marks
- Reverse the Linked List
- Delete All Records
- Save Records to a File

## Programming Language
C

## Data Structure
Singly Linked List

## How to Compile
gcc main.c -o student

## How to Run
./student
